letterboxd-ics

Export Letterboxd diary to iCalendar format

Download .zip Download .tar.gz View on GitHub

letterboxd-ics is a basic script for exporting a Letterboxd to iCalendar format.

How to use

  • Clone the repository:
git clone https://github.com/starsquare/letterboxd-ics.git
cd letterboxd-ics
curl -sS https://getcomposer.org/installer | php
php composer.phar install
  • Edit config.ini to contain your Letterboxd credentials.
  • Point a Web server at the public folder.
  • Enjoy!
